# 🎬 Netflix Clone (React)

A responsive **Netflix UI Clone** built using **React + Vite** that fetches real movie data from TMDB API.
The project replicates core Netflix UI features like movie rows, trailers, search, and responsive layout.

# ⚙️ Installation Guide

Clone the repository

git clone https://github.com/your-username/NetflixClone-React.git

Move into project folder

cd NetflixClone-React

Install dependencies

npm install

Run the development server

npm run dev

Open in browser

http://localhost:5173

---

# 🔑 Environment Setup

Create a `.env` file:

VITE_TMDB_API_KEY=your_tmdb_api_key

---

# 📡 API Used

This project uses **TMDB API** to fetch movie data.

Base URL:

https://api.themoviedb.org/3

Example Endpoint:

/movie/popular
